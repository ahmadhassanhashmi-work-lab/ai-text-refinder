# AI Text Refinement Interface

A frontend-first AI-powered text refinement tool with tone control.

## Features
- Capsule input UI
- Live word counter
- Tone selection
- AI refinement (Gemini via Netlify Functions)
- Dark, modern UX

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Netlify Functions
- Gemini API

## Purpose
Portfolio project demonstrating UI state management and AI integration.
